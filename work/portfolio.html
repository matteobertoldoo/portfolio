About This Project

This portfolio website is a personal project created to showcase my skills, projects, and experience in web development. It serves as an interactive resume, allowing visitors to learn more about me, my services, and my past work.

Check out the source code on GitHub: GitHub Repository

Technologies Used

  • HTML5 – The backbone of the website, providing a clean and semantic structure.
  • CSS3 – Styled using custom CSS and responsive design principles for a modern look.
  • JavaScript – Adds interactivity, including a responsive navigation menu.
  • FontAwesome – Used for icons to enhance the visual appeal.
  • Google Fonts – Typography powered by "Source Code Pro" and "Source Sans Pro".

Challenges & Solutions

During the development of this portfolio, I encountered several challenges, including:

  • Responsive Design: Ensuring that the website looks great on all devices, from desktops to smartphones.
  • Navigation Optimization: Implementing a mobile-friendly menu using JavaScript.
  • Performance: Optimized images and minimized CSS/JS files for faster load times.

Future Improvements

While the portfolio is fully functional, I plan to enhance it with additional features, like a dark mode toggle for better user experience.
